# Responsive Landing Page

## 📌 About the Project

This is a responsive landing page developed using HTML and CSS. The project demonstrates modern web design techniques including Flexbox, CSS Grid, and Media Queries to create a clean, responsive, and user-friendly interface.

## 🚀 Features

- Responsive Navigation Bar
- Hero Section
- Features Section
- Flexbox Layout
- CSS Grid Layout
- Mobile Responsive Design
- Clean and Modern UI

## 🛠 Technologies Used

- HTML5
- CSS3
- Flexbox
- CSS Grid
- Media Queries

## 📂 Project Structure

```
responsive-landing-page/
│
├── index.html
├── style.css
└── README.md
```

## 📱 Responsive Design

The landing page automatically adjusts its layout for different screen sizes using CSS Media Queries.
